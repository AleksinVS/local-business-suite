# Task acceptance: memory MVP simplification

Дата: 2026-05-21.

## Acceptance checks

- `memory.remember` возвращает `memory_id` как главный результат: принято.
- Обычный remember path не создает `MemoryBelief`: принято.
- Обычный remember path не создает `MemoryClaim`: принято для MVP, объект оставлен для будущих этапов.
- `memory.search` не возвращает `memory_belief`: принято.
- Векторный, полнотекстовый и текущий графовый поиск сохранены: принято.
- Текущая обработка очереди не называется полноценной ночной рефлексией: принято.
- Статусы надежности источников сведены к `trusted`, `review_required`, `blocked`: принято.
- Совместимость со старыми `candidate_only`, `quarantined`, `blocked`, `trusted` сохранена: принято.
- Документация и контракты обновлены: принято.

## Остаточные риски

- `MemoryClaim` и `MemoryBelief` остаются физически в схеме ради безопасной совместимости; отдельное удаление возможно только через будущую миграцию с анализом данных.
- Подробная стратегия графового поиска не проектировалась в этом блоке и остается отдельной задачей.
- Команда `memory_reflect_chats` сохранена под старым именем ради совместимости расписаний; новое имя можно добавить отдельным малым изменением.
