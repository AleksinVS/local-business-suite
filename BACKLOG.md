## GUI и справочная система

GUI для настройки системы памяти (настройки api LLM, параметров памяти/обработки/извлечения, каталога настройки прав досутупа, анонимизации и т.д.) и управления пользователями (создание/удаление/отключение/настройка прав доступа). По возможности нужно, чтобы все необходимые настройки можно было делать в GUI. Для каждой настройки предусмотри всплывающую подсказку при наведении и справочную систему с ИИ-ботом - у каждого значимого пункта сделай пиктограмму для открытия плавающего окна миничата - в нем сразу выводится справка по этой настройке и есть окно ввода вопроса по этой настройке, при вводе вопроса ИИ-бот должен сразу понимать о какой настройке идет речь.
Создай необходимую для этого проектную и исполнительскую документацию, при необходимости задавай уточняющие вопросы.
