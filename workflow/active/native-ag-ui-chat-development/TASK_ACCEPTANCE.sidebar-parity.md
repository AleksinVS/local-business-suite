# Task acceptance: sidebar parity

## Решение

Task packet `05-sidebar-history-model-and-clear-parity` принят технически.

## Проверено

- Native sidebar восстанавливает последние сообщения активной sidebar-сессии из `/ai/ui/config/`.
- Сообщения имеют отображаемое время.
- Выбор модели использует общий список `LOCAL_BUSINESS_AI_MODELS`.
- Смена модели обновляет `ChatSession.metadata.model_id`.
- Очистка текущего sidebar-чата удаляет сообщения и `sidebar_summary`, но сохраняет выбранную модель.
- Старый HTMX endpoint очистки использует тот же сервис и не потерял совместимость.
- UI показывает переход в полный чат текущей сессии.

## Ограничения

- Это не закрывает native full-page parity.
- Rich input, Markdown, slash-команды и вложения остаются отдельным срезом.
- Версия AG-UI не обновлялась: актуальная npm-версия выше закрепленной, но обновление требует согласования владельца.
