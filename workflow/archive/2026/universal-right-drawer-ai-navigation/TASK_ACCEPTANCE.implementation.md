# Task Acceptance: implementation

Дата: 2026-05-28.

## Приемка

- Общий right drawer host доступен на всех страницах через `base.html`.
- AI tool `ui.open_right_panel` не принимает произвольный URL и возвращает только серверно сформированный descriptor.
- Доступ к объектам проверяется provider-ами через доменные selectors/policies.
- Заявки и записи листа ожидания имеют собственные providers.
- Обычные локальные drawer-сценарии модулей сохранены.
- Открытая через общий сайдбар запись публикует `data-ai-context`, а page context обновляется после загрузки.
- Runtime prompt объясняет, когда вызывать `ui.open_right_panel`, а когда `ui.get_current_context`.

## Остаточные ограничения

- MVP поддерживает только просмотр (`mode=view`).
- E2E-сценарии зависят от наличия учетных данных `E2E_USERNAME/E2E_PASSWORD` и видимых тестовых объектов.
- Универсальные write-режимы, event bus и расширенные алиасы открытия объектов отложены.
