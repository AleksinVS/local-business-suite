# Active plan: упрощение MVP-архитектуры памяти

Статус: реализовано; workflow acceptance добавлен в `workflow/active/memory-mvp-simplification/`.

Дата: 2026-05-21.

## Цель

Упростить MVP системы памяти, чтобы главным объектом сохраненного знания стал `MemoryKnowledgeItem`, а слой `MemoryBelief` был удален или отключен до следующих этапов.

Система должна сохранить поиск по векторам, полнотекстовому индексу и графу, но подробная стратегия использования графа будет оформлена отдельно.

## Объем работ

- Сделать `MemoryKnowledgeItem` главным объектом памяти MVP.
- Убрать `MemoryBelief` из обычного пути запоминания и поиска.
- Сохранить текущие каналы поиска: векторный, полнотекстовый и графовый.
- Не прорабатывать в этом блоке условия эффективного графового поиска.
- Разделить обработку очереди "запомни" и будущую ночную рефлексию.
- Упростить политику надежных источников до понятной MVP-схемы.
- Обновить документы и проверки.

## Не входит

- Перенос сообщений чата в отдельный SQLite-файл.
- Изменение физического хранения истории чата.
- Пересмотр файловых проекций личной и общей памяти.
- Детальная стратегия графового поиска.
- Production UI для проверки знаний.
- Полноценная система противоречий между источниками.

## Порядок реализации

1. Обновить код обработки явного "запомни": главный результат - `MemoryKnowledgeItem`.
2. Убрать создание и использование `MemoryBelief` в обычном пути.
3. Убрать `MemoryBelief` из `memory.search`, счетчиков и тестовых ожиданий.
4. Проверить, нужно ли создавать `MemoryClaim` автоматически; если нет - отключить в обычном пути.
5. Разделить название и назначение обработчика очереди и будущей ночной рефлексии.
6. Упростить доверие источников до `trusted`, `review_required`, `blocked` или добавить совместимое отображение старых значений.
7. Обновить документацию, тесты и проверки.

## Критерии приемки

- `memory.remember` создает/обновляет `MemoryKnowledgeItem`.
- В обычном пути не создается `MemoryBelief`.
- `memory.search` не ищет и не возвращает `MemoryBelief`.
- Векторный и полнотекстовый поиск проходят без регрессий.
- Графовый поиск остается совместимым, но его стратегия не расширяется в этом блоке.
- Ненадежные источники не попадают в обычный ответ бота.
- Документация ясно говорит, что `MemoryBelief` перенесен на следующие этапы.
- Есть проверки уровня моделей, инструментов памяти и сквозной проверки.
