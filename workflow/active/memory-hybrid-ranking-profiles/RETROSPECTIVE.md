# Retrospective: memory hybrid ranking profiles

## What Worked

- RRF позволил объединить FTS и vector без калибровки разных score-шкал.
- Source semantic search удалось включить через существующий `memory.search`, без нового API.
- Явные профили упростили подсказки ИИ-бота и сделали поведение тестируемым.

## What To Watch

- Document-level vector ухудшит качество для длинных файлов; следующий этап должен перейти к крупным фрагментам.
- Локальные индексы теперь могут содержать PII по принятому решению. Нужно не расширять это поведение на cloud embeddings.
- `pii_audit` требует пользовательского/admin UI для реальной очереди аудита.
