# Настольное приложение уведомлений

## Назначение

Настольное приложение уведомлений - это опциональный Tauri-клиент для Windows и Linux. Оно работает из трея, получает уведомления от портала через внутренний device API и открывает ссылку в системном браузере.

Приложение не заменяет портал и не хранит бизнес-данные заявок. Сервер остается источником прав доступа, текста уведомлений и состояния прочтения.

## Подключение

1. Войдите в портал.
2. Откройте `Уведомления` -> `Устройства`.
3. Нажмите `Сформировать код`.
4. Откройте настольное приложение уведомлений.
5. Укажите адрес портала, например `https://portal.example.local`.
6. Введите одноразовый код подключения.
7. Нажмите `Подключить`.

Код действует 10 минут и используется один раз. После подключения приложение получает ограниченный device token.

## Работа с уведомлениями

- Приложение периодически запрашивает новые уведомления.
- Системное уведомление содержит только короткий безопасный текст.
- Нажатие на системное уведомление или уведомление в окне приложения открывает портал в системном браузере.
- Если портал временно недоступен, приложение увеличивает интервал повторных запросов.
- Если токен отозван, приложение перестает получать уведомления.

## Автозапуск

В окне приложения есть кнопка включения или выключения автозапуска. Поведение зависит от операционной системы и политик рабочего места.

## Отключение устройства

Отключить клиент можно двумя способами:

- в приложении нажать `Отключить`;
- в портале открыть `Уведомления` -> `Устройства` и нажать `Отозвать`.

После отзыва токен больше не работает. Для повторного подключения нужен новый одноразовый код.

## Ограничения

- Для защиты device token нужен HTTPS или другой утвержденный защищенный канал до портала.
- На Linux tray и системные уведомления зависят от окружения рабочего стола.
- На Windows production-распространение потребует доверенного установщика и, желательно, подписи.
- В текущем срезе нет автообновления клиента.
- Возможны дубли с PWA-уведомлениями, если одновременно открыты портал/PWA и Tauri-клиент.
